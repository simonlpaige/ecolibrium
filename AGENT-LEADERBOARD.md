# Agent Leaderboard

> Public credit for AI-assisted contributions to Commonweave.
>
> This is a public signal, not a competitive ranking. The point is not to rank agents; it's to make the work visible and reproducible.

If you're an agent (or human-with-agent) who contributed, your PR will be added here on merge. There are no prizes beyond credit and the PR itself.

---

## Format

Each entry includes:

- **Date** — when the PR was merged
- **Contributor** — agent system + human collaborator (e.g. "Claude Sonnet 4.6 + @username", or "Codex + @username")
- **Track** — Global Directory / Map Intelligence / Edge Provenance / Falsification / Governance Stress-Test / Other
- **Task** — task ID from `AGENT-TASKS.json` if applicable
- **PR** — link
- **Score** — score from `evals/score_pr.py` (when scoring is in place)
- **Summary** — one line on what changed
- **Next task surfaced** — what the contributor left for the next person

## Notes

- Score is informational only. It is not a merge gate.
- Anonymity is fine: if a contributor doesn't want their name on the leaderboard, the entry will list "anonymous" with a contribution-hash for verification.
- Contributions that were later reverted (for capture, false data, etc.) are kept on the leaderboard with a `[reverted]` marker and a brief reason.
