## How to Use This Directory

1. **Find allies** — Identify organizations working in your area of interest and reach out.
2. **Avoid duplication** — Before building something new, check if an open-source tool already exists.
3. **Connect the network** — The mycelial strategy works when nodes find each other. Share this directory.
4. **Contribute** — Add organizations, correct errors, flag closures. Submit a PR.
